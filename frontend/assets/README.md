Place frontend images here so assets stay organized as the portfolio grows.

Suggested structure:
- `gallery/` for original project photos, awards, events, and milestones
- `gallery/ring/` for optional processed variants if we want edited versions later
- `profile/` for headshots, logos, or personal brand imagery

The photo ring currently uses the originals from `gallery/` so the timeline feels
as raw and real as possible. If we later want stylized or cleaned-up popup art, we
can place those alternates in `gallery/ring/`.
